## 2026-02-21 — GitHub Actions Release Pipeline

### Added
- `.github/workflows/release.yml` — CI/CD workflow that triggers on `v*.*.*` tag push.
- `.gitignore` — Excludes compiled binaries and `.env` files from git.

### Workflow Behaviour
- Runs on `ubuntu-latest` GitHub runner.
- Cross-compiles both skills for 4 platforms in a single job (no matrix needed — Go handles cross-compilation natively).
- Uploads 8 binaries to a GitHub Release automatically via `softprops/action-gh-release@v2`.

### Platforms Built (8 binaries per release)
| Suffix | Platform |
|--------|----------|
| `-linux-amd64` | Linux x86-64 |
| `-linux-arm64` | Linux ARM64 / Raspberry Pi |
| `-darwin-arm64` | macOS Apple Silicon |
| `-windows-amd64.exe` | Windows x86-64 |

### How to Release
```bash
git tag v1.x.x
git push origin v1.x.x
```
Release appears at: https://github.com/baladengale/claude-skills/releases
